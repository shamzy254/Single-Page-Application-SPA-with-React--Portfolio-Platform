# Creative Agency Portfolio

A modern Vite + React portfolio platform for a creative agency. This SPA includes:

- A landing page with a project showcase
- A dynamic search feature
- A form for adding new projects
- Responsive styling inspired by modern design mock-ups

## Setup

Install dependencies:

```bash
npm install
```

Run the app locally:

```bash
npm run dev
```

Open the local URL displayed in your terminal.

## Features

- Search projects by title, description, or tags
- Add new portfolio items dynamically
- Responsive layout for desktop and mobile
- Reusable component structure for easy maintenance

## Project structure

- `src/main.jsx` — app entry point
- `src/App.jsx` — main page and state management
- `src/components/` — reusable UI components
- `src/data/projects.js` — initial project data
- `src/styles.css` — global styling
